# 章节编辑器

NovelStack 内置 Markdown 编辑器，支持沉浸式写作体验。

## 编辑器特性

- **Markdown 写作** - 完整 Markdown 语法支持
- **实时预览** - 即时查看渲染效果
- **自动保存** - 每 30 秒自动保存草稿
- **字数统计** - 实时显示字数和阅读时长
- **键盘快捷键** - 高效写作操作

## Markdown 扩展

除了标准 Markdown 语法，编辑器还支持：

### 自定义容器

```markdown
::: info
这是一个信息提示框
:::

::: warning
这是一个警告提示框
:::

::: tip
这是一个小贴士
:::
```

### 代码高亮

使用 Shiki 进行语法高亮：

````markdown
```typescript
const hello = "world";
```
````

### 表格

```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| A   | B   | C   |
```

## 自动保存

编辑器每 30 秒自动保存草稿到本地存储：

- 草稿保存在 `localStorage` (key: `inkweave-draft`)
- 意外关闭时可恢复
- 支持离线编辑

## 内容限制

| 限制 | 值 |
|------|-----|
| 最大内容长度 | 1,000,000 字符 |
| 最大文件上传 | 10 MB |
| 允许的图片类型 | JPEG, PNG, GIF, WebP |

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+B` | 加粗 |
| `Ctrl+I` | 斜体 |
| `Ctrl+K` | 插入链接 |
| `Ctrl+S` | 手动保存 |
| `Ctrl+Shift+P` | 切换预览 |
