# 贡献指南

感谢你考虑为 NovelStack 贡献！

## 行为准则

请保持尊重和建设性的交流。

## 如何贡献

### 报告 Bug

1. 在 GitHub Issues 中搜索是否已有相同问题
2. 如果没有，创建新 Issue
3. 包含：复现步骤、预期行为、实际行为、截图/日志

### 提交功能请求

1. 在 Issues 中描述功能及其使用场景
2. 说明为什么这个功能对 NovelStack 有价值

### 提交代码

1. **Fork** 仓库
2. **创建分支**: `git checkout -b feat/your-feature`
3. **编写代码和测试**
4. **提交**: 使用 [Conventional Commits](https://www.conventionalcommits.org/)
5. **推送**: `git push origin feat/your-feature`
6. **创建 Pull Request**

## Commit 规范

```
feat: 新功能
fix: 修复
docs: 文档
style: 代码格式
refactor: 重构
perf: 性能优化
test: 测试
chore: 构建/工具
```

## Pull Request 要求

- [ ] 代码通过 TypeScript 类型检查
- [ ] 代码通过 ESLint 检查
- [ ] 新功能包含使用文档
- [ ] PR 描述清晰说明改了什么、为什么改
