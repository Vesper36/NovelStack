# 作品管理

作品是 InkWeave 的核心创作单元。每个作品包含卷、章节、标签、分级等完整元数据。

## 作品属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `title` | string | 作品标题（必填） |
| `slug` | string | URL 友好标识符（自动生成） |
| `description` | string | 作品简介 |
| `coverUrl` | string | 封面图片 URL |
| `category` | string | 分类（奇幻、科幻、现代等） |
| `status` | enum | 发布状态 |
| `visibility` | enum | 可见性 |
| `rating` | enum | 内容分级 |
| `wordCount` | number | 总字数（自动计算） |
| `viewCount` | number | 浏览量 |
| `favoriteCount` | number | 收藏数 |

## 作品状态

| 状态 | 说明 | 可见性 |
|------|------|--------|
| `draft` | 草稿 | 仅作者 |
| `reviewing` | 审核中 | 仅作者 |
| `published` | 已发布 | 公开 |
| `archived` | 已归档 | 只读 |

## 可见性设置

| 可见性 | 说明 |
|--------|------|
| `public` | 公开，所有人可见 |
| `private` | 私密，仅作者可见 |
| `link` | 仅链接可访问 |

## 内容分级

| 级别 | 标识 | 说明 |
|------|------|------|
| 全年龄 | `general` | 适合所有读者 |
| 青少年 | `teen` | 13 岁以上 |
| 成人 | `mature` | 17 岁以上，可能包含暴力、性暗示等 |
| 限制级 | `explicit` | 18 岁以上，包含露骨内容 |

## 作品结构

```
作品 (Work)
├── 卷 1 (Volume)         -- 第一卷：序章
│   ├── 章节 1 (Chapter)   -- 第1话 [draft|published]
│   ├── 章节 2             -- 第2话
│   └── 章节 3             -- 第3话
├── 卷 2                   -- 第二卷：展开
│   ├── 章节 4
│   └── 章节 5
└── ...
```

每个卷和章节都有独立的排序、状态和字数统计。

## 创作者中心

### 仪表盘 (`/creator`)

- 统计卡片：作品总数、总浏览量、总收藏数、总字数
- 快捷操作：新建作品、查看作品
- 最近作品网格

### 我的作品 (`/creator/works`)

- 列出当前用户的所有作品
- 显示每部作品：状态徽章、可见性、分类、标签、章节数（已发布/总数）、浏览量、收藏数
- 支持按标题/描述/分类/标签搜索筛选
- 汇总统计：作品数、章节数、字数、浏览量

### 新建作品 (`/creator/works/new`)

表单字段：
- 标题（必填）
- 简介
- 分类
- 标签（逗号分隔）
- 内容分级（四级）
- 可见性（公开/私密/仅链接）

### 作品管理 (`/creator/works/[workId]`)

- 元数据编辑：标题、简介、分类、分级、可见性、状态、标签、封面
- 卷管理：创建/编辑/删除卷，展开/折叠
- 章节管理：创建/删除章节，显示字数和状态
- 拖拽排序 UI（功能开发中）

## 作品浏览

### 列表页 (`/works`)

支持多种浏览方式：

| 排序 | 说明 |
|------|------|
| 最新更新 | 按更新时间倒序 |
| 最受欢迎 | 按浏览量倒序 |
| 最长 | 按字数倒序 |

筛选：
- 按标签筛选（侧边栏标签云）
- 按分类筛选
- 文本搜索（标题/简介）

分页：每页 20 部作品

### 详情页 (`/works/[slug]`)

展示作品完整信息：
- 作品元数据（标题、简介、作者、分类、分级）
- 标签列表
- 卷章目录（树形结构）
- 浏览量、收藏数、总字数
- 开始阅读按钮

## 标签系统

### 标签分类

| 分类 | 示例标签 |
|------|----------|
| 题材 | 奇幻、科幻、现代、古代、架空、未来、末日、校园、都市、仙侠 |
| 类型 | 言情、悬疑、恐怖、喜剧、悲剧、冒险、推理、日常、热血、治愈 |
| 配对 | BG、BL、GL、无CP、多CP、逆CP |
| 状态 | 连载中、已完结、暂停、弃坑 |

### 标签浏览 (`/tags`)

- 按分类展示所有标签
- 显示每个标签的使用次数
- 点击标签跳转到作品列表筛选
