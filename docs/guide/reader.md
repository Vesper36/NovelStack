# 阅读器

InkWeave 阅读器提供高度可定制的沉浸式阅读体验，所有设置自动持久化。

## 阅读设置

通过浮动工具栏自定义阅读体验：

| 设置 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 字体大小 | 16px | 14-24px | 调整文字大小 |
| 行高 | 1.75 | 1.2-2.5 | 调整行间距 |
| 阅读宽度 | 65ch | - | 调整内容区宽度 |
| 字体 | 系统字体 | - | 阅读字体选择 |

所有设置通过 Zustand persist 存储在 localStorage，跨页面自动恢复。

## 滚动进度条

页面顶部显示一条细进度条，实时反映当前阅读位置：

- 使用 CSS `scaleX` 变换实现，性能优异
- 滚动时平滑更新
- 颜色跟随当前主题强调色

## 智能工具栏

阅读器底部的浮动工具栏具有智能显隐行为：

- **向下滚动** - 工具栏自动隐藏，最大化阅读空间
- **向上滚动** - 工具栏滑出显示
- **悬浮在底部** - 不遮挡内容

工具栏包含：
- 主题切换按钮（循环切换六款主题）
- 字体大小 +/- 按钮
- 目录开关按钮

## 章节目录 (TOC)

章节目录从 Markdown 标题结构自动生成：

### 桌面端

- 右侧固定侧边栏
- 显示 h2-h3 级标题
- 当前阅读位置高亮
- 点击标题平滑滚动到对应位置

### 移动端

- 底部弹出面板 (Bottom Sheet)
- 点击目录图标展开
- 支持滑动关闭

## 阅读进度

- 自动记忆每章的阅读位置（scroll position）
- 进度百分比计算（基于 scrollHeight）
- 下次打开自动恢复到上次位置
- 进度数据存储在 IndexedDB（`ReadingProgressStore`）

## 章节导航

- **底部导航** - 上一章 / 下一章按钮
- **键盘快捷键** - 左右箭头键切换章节
- **章节目录** - 侧边栏跳转任意章节
- **返回作品页** - 面包屑导航

## 作者有话说

章节正文结束后，展示作者笔记区域。支持 Markdown 格式，与正文样式区分。

## SEO 优化

阅读页面自动注入：

- **OpenGraph 元数据** - og:title, og:description, og:type
- **JSON-LD Book 结构化数据** - 便于搜索引擎理解
- **作品标题和章节标题** - 用于浏览器标签页

## 主题支持

阅读器支持全部六款主题，详见 [主题系统](/guide/themes)。

阅读时按 `T` 键快速循环切换主题。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `T` | 循环切换主题 |
| `Left Arrow` | 上一章 |
| `Right Arrow` | 下一章 |
| `+` | 增大字体 |
| `-` | 减小字体 |
| `Esc` | 关闭弹出面板 |

## 技术实现

阅读器核心组件 `ReadingView` 的实现特点：

- 使用 CSS 变量 (`var(--bg-primary)` 等) 实现主题即时切换
- `IntersectionObserver` 实现滚动方向检测
- `requestAnimationFrame` 优化滚动事件处理
- `framer-motion` 实现工具栏动画
- 章节内容由服务端预编译 HTML，首屏渲染速度快
