# 阅读器

NovelStack 阅读器提供高度可定制的沉浸式阅读体验。

## 阅读设置

通过阅读器工具栏，读者可以自定义：

| 设置 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| 字体大小 | 16px | 14-24px | 调整文字大小 |
| 行高 | 1.75 | 1.2-2.5 | 调整行间距 |
| 阅读宽度 | 65ch | - | 调整内容区宽度 |

设置保存在 `localStorage`，下次阅读自动恢复。

## 阅读进度

- 自动记忆每章的阅读位置
- 下次打开自动跳转到上次位置
- 进度条显示当前章节阅读百分比

## 导航

- **上一章 / 下一章** - 底部导航按钮
- **章节目录** - 侧边栏显示当前作品的卷章结构
- **返回作品页** - 快速返回作品详情页

## 主题支持

阅读器支持全部六款主题：

1. **默认** - 白底黑字，经典阅读体验
2. **深夜** - 暗色护眼
3. **羊皮纸** - 复古纸质风格
4. **赛博朋克** - 科技绿感
5. **森林** - 自然清新
6. **海洋** - 深邃宁静

阅读时可按 `T` 键快速切换主题。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `T` | 切换主题 |
| `← / →` | 上一章 / 下一章 |
| `+ / -` | 调节字体大小 |
| `Esc` | 关闭/返回 |
